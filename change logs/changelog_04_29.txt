BACKEND

routes/stats.py
---------------
ORIGINAL CHANGES
- Lines 74-123 (get_player_statistics): Replaced single-entry statistics read
  (`statistics[0]`) with a full aggregation across all competition entries
  returned by the API for the season (e.g. Premier League + Champions League +
  national team competitions etc.)
- Lines 74-123 (get_player_statistics): Fixed API-Football typo — the API
  returns "appearences" (misspelled) instead of "appearances"; the old code
  used the correct spelling so it always got None → 0, causing stats_available
  to be False for every player regardless of actual data

NEW ADDITIONS
- Lines 74-110 (_sum_player_stats): New helper function that iterates over all
  competition entries in a player's statistics list and returns summed totals
  for goals, assists, appearances, yellow cards, red cards, as well as the
  player's primary position and current team name; handles both the API typo
  ("appearences") and the correct spelling ("appearances") for forward
  compatibility

ORIGINAL CHANGES
